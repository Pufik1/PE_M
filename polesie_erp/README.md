# ERP система ОАО "Полесьеэлектромаш"

Корпоративная система управления производственным предприятием.

## 📋 Описание

Система предназначена для автоматизации основных бизнес-процессов ОАО "Полесьеэлектромаш":
- Управление продукцией (электродвигатели, насосы, электроконфорки, литье)
- Обработка заказов клиентов
- Управление производственными заданиями
- Складской учёт
- Работа с контрагентами
- Отчётность и аналитика

## 🛠 Технологии

- **Backend:** PHP 7.4+
- **Database:** MySQL / MariaDB
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Шрифты:** Google Fonts (Inter)
- **Сервер:** MAMP / Apache + Nginx

## 📦 Установка

### 1. Требования
- MAMP Pro или аналогичный стек (Apache/Nginx + MySQL + PHP)
- PHP 7.4 или выше
- MySQL 5.7 или выше

### 2. Настройка базы данных

1. Запустите MAMP и откройте phpMyAdmin
2. Создайте новую базу данных `polesie_erp`
3. Импортируйте файл `database.sql`:
   - Откройте phpMyAdmin
   - Выберите базу данных `polesie_erp`
   - Перейдите на вкладку "Импорт"
   - Выберите файл `database.sql`
   - Нажмите "Вперёд"

### 3. Конфигурация приложения

Откройте файл `config.php` и проверьте настройки подключения:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'polesie_erp');
define('DB_USER', 'root');
define('DB_PASS', 'root'); // Стандартный пароль MAMP
```

### 4. Размещение файлов

Скопируйте все файлы проекта в директорию веб-сервера:
- Для MAMP: `/Applications/MAMP/htdocs/polesie_erp/`
- Или настройте виртуальный хост

### 5. Запуск

Откройте в браузере: `http://localhost/polesie_erp/`
