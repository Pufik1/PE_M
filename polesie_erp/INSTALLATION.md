# Инструкция по исправлению ошибок и добавлению функционала

## 1. Исправление ошибки "Unknown column 'comment' in 'field list'"

### Причина ошибки
В таблице `orders` отсутствовало поле `comment`, которое использовалось в API при обновлении заказа.

### Решение
Выполните один из следующих вариантов:

**Вариант A: Обновить существующую базу данных**
```sql
USE polesie_erp;
ALTER TABLE orders ADD COLUMN IF NOT EXISTS comment TEXT;
```

**Вариант B: Использовать готовый SQL-файл**
Запустите файл `update_database.sql` через phpMyAdmin или командную строку MySQL:
```bash
mysql -u root -p < update_database.sql
```

**Вариант C: Пересоздать базу данных**
Если вы только начинаете работу с системой, просто импортируйте обновлённый файл `database.sql` - поле `comment` уже включено в структуру таблицы `orders`.

## 2. Добавление функции создания нового заказа

### Новые файлы
- `/api/create_order.php` - API endpoint для создания заказов

### Изменённые файлы
- `/orders.php` - Добавлена кнопка "Добавить заказ" и модальное окно
- `/database.sql` - Добавлено поле `comment` в таблицу `orders`

### Как использовать
1. Откройте страницу "Заказы" (`orders.php`)
2. Нажмите кнопку "Добавить заказ" в правом верхнем углу таблицы
3. Заполните форму:
   - **Номер заказа** (обязательно) - например, ORD-2024-100
   - **Клиент** - выберите из списка контрагентов
   - **Статус** - начальный статус заказа
   - **Сумма** - общая сумма заказа в BYN
   - **Комментарий** - дополнительная информация
4. Нажмите "Сохранить"

### Структура модального окна
Модальное окно поддерживает два режима:
- **Создание** - форма отправляется на `api/create_order.php`
- **Редактирование** - форма отправляется на `api/update_order.php`

## 3. Проверка работы

1. Убедитесь, что поле `comment` добавлено в базу:
```sql
DESCRIBE orders;
```

2. Проверьте создание заказа через интерфейс:
   - Создайте новый заказ
   - Проверьте, что он появился в таблице
   - Попробуйте отредактировать заказ

3. Проверьте API напрямую (опционально):
```bash
curl -X POST http://localhost/polesie_erp/api/create_order.php \
  -d "order_number=TEST-001&status=new&total_amount_byn=1000&comment=Тестовый заказ"
```

---

## 4. Примечания

- Все изменения совместимы с существующей структурой базы данных
- Поле `comment` теперь доступно как при создании, так и при редактировании заказа
- Модальное окно автоматически подгружает данные при редактировании через `api/get_order.php`
